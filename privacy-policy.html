Privacy Policy

OmniStudio DR/IP Tracer does not collect, store, transmit, or share personal or sensitive user data.

Data Usage

This extension reads OmniStudio DataRaptor and Integration Procedure request and response data locally inside the browser for debugging purposes only.

Storage

Trace logs are temporarily stored using Chrome local storage and are cleared automatically or manually by the user.

Third Parties

No data is transmitted to external servers or shared with third parties.

Permissions

This extension uses Salesforce domain access only to monitor OmniStudio network calls.
